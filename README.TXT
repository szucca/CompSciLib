PURPOSE OF PROJECT: A library to help AP Computer Science Students

VERSION or DATE: August 2015
HOW TO START THIS PROJECT:
AUTHORS: S Zucca

USER INSTRUCTIONS:
The .jar file should be installed on all of the school computers running BlueJ. This makes the classes available for import using

import org.brookfieldps.UserInput;

Then, input can be collected through the methods UserInput.getString(), UserInput.getInter(), etc.
